### JDBC - Java Database Connectivity

- Java언어와 DB를 연결해주는 통로와 같은 것.
- Java를 이용한 <b>DB접속</b> 과 <b>SQL문장의 실행</b>
- 실행 결과로 얻어진 데이터의 핸들링을 제공하는 방법과 절차에 관한 규약.
- Java 프로그램내에서 SQL문을 실행하기 위한 자바 API.
- SQL과 프로그래밍 언어의 통합 접근 중 한 형태
